# Serbian Prefixes through the Prism of Distributional Semantic Models

Репозиторий содержит материалы к выпускной квалификационной работе
«Исследование сербских приставок с помощью моделей дистрибутивной семантики».

В работе исследуется сербская глагольная префиксация. Основная идея состоит в
том, чтобы проверить, можно ли описывать приставки как регулярные семантические
преобразования в векторном пространстве. Для этого используются собственные
векторные представления сербских глаголов и матрицы преобразования, которые
предсказывают вектор приставочного глагола по вектору базового глагола.

## Структура репозитория

- `scripts/` - основные скрипты для подготовки данных, обучения моделей и оценки результатов.
- `notebooks/` - исследовательские ноутбуки и итоговый датасет глагольных пар.
- `data/` - исходные и промежуточные данные, использованные при подготовке корпуса и датасета.
- `csv/` и `json/` - промежуточные таблицы, полученные на разных этапах сбора и фильтрации данных.
- `results/` - результаты экспериментов, отчеты, графики и материалы, использованные при написании работы.
- `коды для создания датасета/` - ноутбуки и скрипты, связанные с формированием датасета.

## Основные данные

Главный датасет для экспериментов:

```text
notebooks/serbian_verb_final_hf_wiki_5_ruscorpora_combined_50.csv
```

Он содержит пары вида:

```text
base_verb,prefixed_verb,prefix,freq_base,freq_prefixed
```

В работе используется 1497 пар сербских глаголов. Для каждой пары указаны
базовый глагол, приставочный глагол, приставка и частотные характеристики.

## Основные скрипты

### Обучение собственных эмбеддингов

```text
scripts/train_own_serbian_embeddings.py
scripts/train_own_serbian_lemma_embeddings.py
```

Эти скрипты используются для обучения собственных моделей Word2Vec и FastText
на сербском корпусном материале. В работе сравниваются три варианта
представлений:

- FastText на локальном корпусном материале;
- Word2Vec на лемматизированных данных;
- FastText на лемматизированных данных.

### Обучение матриц приставок

```text
scripts/train_prefix_matrices.py
scripts/train_prefix_matrices_own_representations.py
```

Скрипты обучают линейные преобразования для приставок. Общая идея модели:

```text
v(prefixed_verb) ~= v(base_verb) @ W_prefix
```

То есть для каждой приставки обучается матрица, которая переводит вектор
базового глагола в пространство приставочного глагола.

### Оценка результатов

```text
scripts/evaluate_prefix_matrices.py
scripts/make_embedding_comparison_report.py
scripts/make_own_fasttext_report.py
```

Эти скрипты используются для оценки качества матриц и сравнения разных моделей
эмбеддингов. Основная метрика - косинусная близость между предсказанным и
реальным вектором приставочного глагола.

Итоговые таблицы и отчеты находятся в папке:

В частности, там сохранены результаты сравнения собственных моделей

## Примечание о воспроизводимости

Часть больших файлов, например обученные модели эмбеддингов и некоторые
корпусные материалы, может не входить в репозиторий из-за размера. Они
создаются локально при запуске скриптов и обычно сохраняются в папки
`models/embeddings/` и `models/artifacts/`.

Для запуска основных скриптов нужны Python 3 и библиотеки для работы с данными
и векторными моделями, в частности:

```text
pandas
numpy
scikit-learn
gensim
matplotlib
```
